# Titanic Survival Prediction

## Project Overview

This project predicts Titanic passenger survival using machine learning classification models. The analysis includes exploratory data analysis, missing value handling, feature engineering, categorical encoding, model training, hyperparameter tuning, and model comparison.

## Dataset

The project uses the Titanic dataset with passenger information such as age, sex, passenger class, fare, family size, ticket information, and survival status.

## Project Workflow

1. Data loading and inspection
2. Missing value analysis
3. Outlier detection
4. Feature engineering
5. Categorical encoding
6. Train-test split
7. Model training
8. Hyperparameter tuning with GridSearchCV
9. Model comparison

## Models Used

- Logistic Regression
- Support Vector Classifier
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors Classifier


## Project Status

Completed as a machine learning classification practice project.

## Files

- `titanic_dataset.ipynb`: Main notebook containing the full analysis and modeling process.
- `train.csv`: Training dataset.

## Conclusion

This project demonstrates a complete machine learning workflow for a binary classification problem. Multiple classification algorithms were compared to predict passenger survival.
